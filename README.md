# NKU-GateWay
南开网关登录脚本


## 单文件

下载 [single.py](https://raw.githubusercontent.com/NewFuture/NKU-Gateway/master/single.py)

### 登录
```
python single.py 
```

### 注销
```
python single.py logout
```

### 循环登录[断网重连]

```
python single.py loop
```


## 多文件【会改代码的用这个】


自动登录(掉线重连)
修改autologin.py
第七行八行的账号和密码
```
python autologin.py
```

## 注销
```
python logout.py
```
